ai GPT 生成网文

Created2022-07-07|Updated2022-07-07
|Post View:
  • 原作者github地址：https://github.com/BlinkDL/AI-Writer

目录结构

依赖

paddlepaddle>=2.2 请参考 paddle安装教程 安装paddlepaddle2.2+

命令行使用方法

    1. 进入原作者的 https://github.com/BlinkDL/AI-Writer ，下载模型，在右边 Release（v2022-02-15 最新综合模型（N卡）），解压 pth 到 model 目录。
    1. python convert_pytorch2paddle.py 转换权重。
    1. 运行 python run.py。

网页界面使用方法：

    1. 进入原作者的 https://github.com/BlinkDL/AI-Writer ，下载模型，在右边 Release（v2022-02-15 最新综合模型（N卡）），解压 pth 到 model 目录。
    1. python convert_pytorch2paddle.py 转换权重。
    1. 运行 python server.py 开启服务。
    1. 打开 web-client/index.html，进行愉快的玩耍。
